# Project: Stride Planner
A digital task prioritization app focused on daily progress tracking.

## Tech Stack
- Framework: Next.js (App Router)
- Animations: Framer Motion
- Icons: Lucide-react
- Celebration: canvas-confetti

## Design System (The "Zen Rings")
- **Positions:** Concentric rings located on the right side of the screen.
- **Color Logic:** - Navy Blue = "Tasks"
  - Sage Green = "Priorities"
- **Daily Stats:** Includes steps and water tracking near the rings.

## Coding Standards
- **Lists:** Use letters (a, b, c) for sub-tasks when 'Tab' is hit.
- **Visual Rewards:** Show "Daily Goal Met!" badge next to steps when target reached.
- **Components:** Functional components with Tailwind CSS.
